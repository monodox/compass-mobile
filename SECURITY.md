# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.x     | ✅        |

## Reporting a Vulnerability

If you discover a security vulnerability, please do **not** open a public issue.

Instead, report it privately by emailing: **security@compass.app**

Include:
- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fix (optional)

We aim to respond within **48 hours** and will work with you to resolve the issue promptly. Once fixed, we'll credit you in the release notes unless you prefer to remain anonymous.
